Descrição

  Permite realizar operações aritméticas sobre os pixels da imagem.
  Para tanto deve ser informado um valor inteiro para GAIN e para OFFSET.

Passos

  1 - Informar um valor inteiro de GAIN.
  2 - Informa um valor inteiro de OFFSET.
  3 - Confirmar a operação - pressionar botão OK
  ou
  3 - Cancelar a operação - pressionar o botão CANCELAR
